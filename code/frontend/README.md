# Streamlit前端使用说明

## 项目结构

```
frontend/
├── main.py                          # 主入口文件
├── config.py                        # 配置文件
├── requirements.txt                 # 前端依赖
├── utils/
│   ├── api_client.py               # API客户端
│   └── helpers.py                  # 工具函数
└── pages/
    ├── login.py                    # 登录页面
    ├── data_management.py          # 数据管理页面
    ├── model_management.py         # 模型管理页面
    ├── predict_management.py       # 预测管理页面
    └── admin.py                    # 系统管理页面
```

## 安装步骤

### 1. 安装依赖
```bash
cd frontend
pip install -r requirements.txt
```

### 2. 确保后端服务已启动
后端Flask服务需要在 http://localhost:5000 运行

### 3. 启动Streamlit前端
```bash
streamlit run main.py
```

前端将在 http://localhost:8501 启动

## 功能说明

### 登录页面
- 用户登录
- 密码重置（通过安全问题）
- 测试账号提示

### 数据管理（数据分析师）
- **数据查询**：按时间范围查询负荷数据，支持分页和导出
- **数据上传**：支持CSV批量上传和手动输入
- **异常检测**：Z-score、IQR、箱型图三种方法
- **数据处理**：缺失值填充、数据归一化、相关性分析
- **数据可视化**：负荷曲线图表生成

### 模型管理（模型工程师）
- **模型配置**：创建LSTM、Transformer、混合模型配置
- **模型训练**：设置训练参数，执行模型训练
- **训练记录**：查看历史训练记录和状态
- **模型版本**：创建版本、激活版本、性能对比

### 预测管理（业务用户）
- **创建任务**：选择模型版本，设置预测时间范围
- **任务列表**：查看和执行预测任务
- **预测结果**：查看结果数据，更新实际值，计算误差
- **结果分析**：预测对比图、散点图、可视化分析

### 系统管理（系统管理员）
- **用户管理**：创建、编辑、删除用户，权限管理
- **操作日志**：查看系统操作日志
- **告警管理**：查看和处理系统告警
- **系统配置**：查看系统配置参数

## 注意事项

1. **后端依赖**：前端必须在后端服务启动后才能正常使用
2. **API地址**：默认后端地址为 http://localhost:5000，可在config.py中修改
3. **会话管理**：使用Streamlit的session_state管理登录状态
4. **图表显示**：后端返回base64编码的图片，前端直接显示
5. **数据导出**：支持CSV和JSON格式导出
6. **权限控制**：不同角色只能访问对应的功能页面

## 常见问题

### 1. 无法连接到服务器
- 检查后端Flask服务是否启动
- 确认后端地址配置正确（config.py中的API_BASE_URL）

### 2. 登录失败
- 检查用户名密码是否正确
- 确认用户状态是否为启用

### 3. 图表不显示
- 检查后端是否正确返回base64编码的图片
- 确认matplotlib字体配置正确

### 4. 页面刷新后需要重新登录
- 这是Streamlit的正常行为，session_state在页面刷新后会重置

## 技术栈
