# ⚡ 基于 LSTM-Transformer 的城市电网短期负荷预测系统

## 项目概述

本系统是一套面向城市电网调度场景的**短期电力负荷预测平台**，以深度学习模型（LSTM、Transformer、LSTM-Transformer 混合）为核心预测引擎，配套完整的数据管理、模型训练、预测任务和系统运维功能，提供可直接使用的 Web 交互界面。

---

## 核心功能模块

### 1. 用户权限管理
- 四种角色：系统管理员 / 数据分析师 / 模型工程师 / 业务用户
- 基于角色的菜单权限隔离
- 用户登录、密码重置（安全问题验证）、账号启用/禁用

### 2. 数据管理（数据分析师）
- **数据上传**：支持标准 CSV（`recordtime` / `loadvalue`）、COMED 格式（`Datetime` / `COMED_MW`）和 ETT-small 格式（`date` / `OT`）自动转换，超 5000 条自动分批上传
- **数据查询**：分页查询，按时间范围过滤，支持下载 CSV
- **异常检测**：Z-score、IQR、箱型图三种方法，可视化异常点分布
- **数据处理**：线性/样条缺失值填充、Min-Max 归一化、特征相关性分析
- **数据可视化**：时序负荷曲线图

### 3. 模型管理（模型工程师）
支持三种深度学习模型架构：

| 模型 | 结构说明 |
|------|----------|
| **LSTM** | 多层 LSTM + 全连接输出层，hidden_size/num_layers/dropout 可配 |
| **Transformer** | 多头自注意力编码器，d_model/nhead/num_layers 可配 |
| **LSTM-Transformer 混合** | LSTM 提取局部时序特征 → Transformer 建模全局依赖 → 全连接输出 |

- 模型超参数 JSON 可视化配置
- 训练/验证集自动按数据库时间范围切分（默认 80% / 20%）
- 训练过程记录（loss、状态、耗时）
- 模型版本管理，支持激活/切换版本

### 4. 预测管理（业务用户）
- 创建预测任务，选择激活模型版本，指定预测时间范围
- 以 6 小时为步长自动生成预测序列
- 预测结果表格查看与 JSON 导出
- 更新实际值后计算误差指标：MAE / RMSE / MAPE / R²
- 预测值 vs 实际值对比可视化

### 5. 系统管理（管理员）
- 用户管理：增删改查、角色分配、启用/禁用
- 操作日志：分页查询全量操作记录
- 告警管理：多级别告警（info / warning / error / critical）处理
- 系统配置：键值对在线编辑

## 快速启动

### 1. 安装依赖
```bash
pip install -i https://mirrors.aliyun.com/pypi/simple/ \
    Flask==2.3.0 Flask-SQLAlchemy==3.0.5 PyMySQL==1.1.0 \
    numpy pandas scikit-learn matplotlib seaborn scipy \
    streamlit==1.28.0 plotly torch==2.0.1
```

### 2. 初始化数据库
```bash
# 在 MySQL 中执行
mysql -u root -p < database.sql
```

### 3. 启动前端（Streamlit）
```bash
cd frontend
python -m streamlit run main.py
```

### 4. 启动后端接口（可选，供外部调用）
```bash
python app.py
```

## 数据集说明

当前系统使用 **ETT-small 数据集**（`ETT-small/ETTh1.csv`），这是电力变压器温度（Electricity Transformer Temperature）数据集，包含约 17420 条**小时级时序数据**，时间跨度 2016-07 至 2018-06。其中 `OT`（Oil Temperature）列作为预测目标值（负荷），导入时自动缩放到 3000–10000 MW 范围。

`data/` 目录下保留的 PJM 电网各区域小时级电力负荷数据（CSV 格式，来源：Kaggle PJM Hourly Energy Consumption）可作为备用数据集。

---

*基于 LSTM-Transformer 混合架构实现电网负荷预测，融合 LSTM 对局部时序规律的提取能力与 Transformer 对长距离依赖的建模能力，提升短期预测精度。*
